Result Interpretation
  • ASM: Allele-Specific Methylation (as est. by methbat).
  • G (gnomAD): External link to gnomAD v4 Browser.
  • N (NCBI): External link to NCBI Genome Data Viewer.
  • QC PASS: both report_summary and qc_warn are 'PASS' (Green).
  • QC Warning: either report_summary or qc_warn is not 'PASS' (Orange).
  • QC Failed: both report_summary and qc_warn are not 'PASS' (Red).
  • meth score: Methylated fraction (as calculated by methbat).
  • unmeth score: Unmethylated fraction (1 - meth score).

How to Load & View Phased Methylation in IGV

Step 2: Navigate Click any table row to copy locus, then paste into IGV search box.
Step 3: Color Gear > Color by > base mod 2-color (5mC)
Step 4: Group Gear > Group by > tag:HP (Separate H1/H2)
Methods and Context

This report summarizes hg38 methylation loci identified by the gsm-nallo pipeline. Methylation is phased using HiPhase and visualized using IGV-Web.

Required Files
  • *methbat.csv methbat results
  • *haplotagged.bam & .bai (Phased Reads)
  • *combined.bw (Total Methylation Signal)
  • *hap1.bw / *hap2.bw (Allele-Specific Tracks)
